на Java 17 и Spring Boot 3.5.0, меняя библиотеки и технологии на ходу. Поэтому готовлюсь много гуглить,

## twitter-to-kafka-service

Микросервис предназначен для стримминга твитов по заданным ключевым словам из социальной сети  
X (ex-Twitter).
Как оказалось, с 2023 года на бесплатном аккаунте получение стриммингового потока недоступно. Возможно
получить данные только через REST-запрос с использованием Bearer-токена, и то с ограничениями:
один запрос в 15 минут, от 10 до 100 результатов в запросе, до 1500 твитов в месяц :angry:  
Тем не менее, конфигурацию я сохранила. Может пригодиться.

В папку resources сервиса `twitter-to-kafka-service` следует поместить конфигурационный файл twitter4j.properties.
Пример файла:
```properties
debug=true
oauth.consumerKey=*********************
oauth.consumerSecret=******************************************
oauth.accessToken=**************************************************
oauth.accessTokenSecret=******************************************
```
Ключи и токены можно получить в аккаунте разработчика социальной сети X (ex-Twitter).
